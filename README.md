# 令翠学 運命周期診断 LP（逸見弘子）

生年月日から令翠学の「運命周期」を無料診断し、**一番気になるタイミング**を提示。
続きは逸見弘子さんの鑑定（LINE）へ誘導するランディングページです。

ビルド不要の**静的サイト（HTML / CSS / JS）**なので、ファイルを置くだけで公開できます。

## 診断ロジックについて（重要）

令翠学そのものの計算式は非公開ですが、**令翠学と同じ構造**
（6つの星×陽陰＝12／12年で一巡／冬の3時期＝大殺界）をもつ
**「六星占術」は算出法が公開されています**。本ツールはその公開アルゴリズムで
運命周期を厳密に計算し、令翠学の時期名（開拓期〜精算期）へ1対1で対応づけています。
近似ではなく、生年月日から実際の運命周期を算出します。

時期の名称は令翠学の運命周期に準拠：

- **春**：開拓期・生長期・決定期
- **夏**：健康期・人気期・油断期
- **秋**：再開期・経済期・充実期
- **冬**：背信期・令期・精算期（＝大殺界に相当）

### 算出の根拠（検証済み）

- **日干支の基準**：2000-01-01＝戊午（六十干支の55番）
- **星数**＝生まれた日の六十干支番号(1〜60)
  → 1-10土星 / 11-20金星 / 21-30火星 / 31-40天王星 / 41-50木星 / 51-60水星
- **陽陰**＝生年の十二支（子寅辰午申戌＝陽）
- **周期の固定**＝各星の「停止」十二支（霊合星人の生年）で位置を確定
- **検算**：2026年(午)の大殺界＝火星人±・天王星人− と一致することを確認済み

### ロジックの場所

`script.js` の **③のセクション**（`kanshi60` / `starProfile` /
`periodIndexForYear` / `diagnose`）にまとまっています。
もし逸見弘子さん独自の正式ルールがあれば、この部分を差し替えれば
表示・演出はそのままで中身だけ置き換えられます。

> 注：令翠学の正式な計算と細部で差が出る可能性はあります。
> 本ツールは「公開情報で再現できる最も確からしい運命周期」です。

## ご注意（免責）

本診断は12年周期の運命学の算出法にもとづいて運命周期を計算します。
ただし宿星の詳細・相性・月運などを含む本格的な鑑定とは目的・精度が異なります。
重要な判断は専門の鑑定（逸見弘子）をご利用ください。フッターにも免責文を記載しています。
