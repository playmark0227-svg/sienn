# 令翠学 運命周期診断 LP（逸見弘子）

生年月日から令翠学の「運命周期」を無料診断し、**一番気になるタイミング**を提示。
続きは逸見弘子さんの鑑定（LINE）へ誘導するランディングページです。

ビルド不要の**静的サイト（HTML / CSS / JS）**なので、ファイルを置くだけで公開できます。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | ページ本体（構成・文言） |
| `styles.css` | デザイン（紫基調・上品） |
| `script.js` | 診断ロジック・結果表示 |

## 公開方法（どれでもOK）

- **ローカル確認**: `index.html` をブラウザで開くだけ。
  （Google Fonts 読み込みのためネット接続を推奨）
- **そのまま設置**: レンタルサーバーに3ファイルをアップロード。
- **無料ホスティング**: Netlify / Cloudflare Pages / GitHub Pages にこのフォルダをそのまま。

ローカルで簡易サーバーを使う場合：

```bash
# このフォルダで
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## 運用前に必ず差し替える項目

### 1. LINE公式アカウントのURL（最重要）

`script.js` の先頭 `CONFIG.LINE_URL` を、逸見弘子さんの友だち追加URLに変更してください。

```js
const CONFIG = {
  LINE_URL: "https://lin.ee/REPLACE_ME", // ← ここを差し替え
  ...
};
```

1か所変えるだけで、ページ内すべてのLINEボタンに反映されます。

### 2. プロフィール・文言

- 逸見弘子さんの肩書き・実績・写真は `index.html` の
  `<!-- 逸見弘子 プロフィール -->` セクションを編集してください。
  （写真を使う場合は `<div class="profile-avatar">…</div>` を `<img>` に置き換え）
- キャッチコピーや説明文も `index.html` 内で自由に調整できます。

## 診断ロジックについて（重要）

令翠学の**正式な算出法は非公開**のため、現在は

> 「運命は12年で一巡する」という令翠学の考え方に基づいた**再現性のある近似**

で現在地と一番気になるタイミングを求めています（同じ生年月日なら毎回同じ結果）。
時期の名称は令翠学の運命周期に準拠しています：

- **春**：開拓期・生長期・決定期
- **夏**：健康期・人気期・油断期
- **秋**：再開期・経済期・充実期
- **冬**：背信期・令期・精算期

### 公式ロジックへの差し替え方

公式の対応表・計算ルールが手に入ったら、`script.js` の
**`periodIndexForAge()`（③のセクション）だけ**を差し替えれば、
表示や演出はそのまま正確な診断にできます。

```js
// 「年齢 age・固有オフセット offset のときの周期 index(0..11)」を返す
function periodIndexForAge(age, offset) {
  // ここを公式ロジックに置き換える
  return (((age + offset) % 12) + 12) % 12;
}
```

## 設定できるその他の項目（`script.js` の `CONFIG`）

| 項目 | 内容 |
| --- | --- |
| `LINE_URL` | LINE友だち追加URL |
| `YEAR_MIN` / `YEAR_MAX` | 生年の選択範囲 |
| `LOADING_MS` | 「鑑定中…」演出の長さ（ミリ秒） |

入力された生年月日・ニックネームは**この端末内（ブラウザ）でのみ計算**され、
